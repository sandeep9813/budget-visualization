# E-Governance Budget Visualization

An interactive dashboard to visualize government budget allocations across sectors and provinces, compare historical budgets, and allow citizens to submit feedback.

## Features
- Compare historical budgets by year.
- Visualize provincial budget allocations by sector.
- Citizen feedback and complaint submission.
- Responsive and interactive charts and tables.

- ## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- Visualization:  Chart.js
